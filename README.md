# Calculo peso ideal de una persona
Este programa trata de calcular cual seria el peso ideal de una persona a partir de unos datos.
###  Datos de entrada
- Nombre
- Sexo
- Altura
- Peso actual
### Datos de salida
- Peso ideal de la persona
- Exceso o defecto de peso
